# 인프라 공방 6기
인프로 공방 6기 메모입니다.

## 그럴듯한 인프라 만들기
### 0단계 - pem 키 생성하기
![](images/00-키페어생성.png)

* AWS Console 접속
* `EC2` -> `네트워크 및 보안` -> `키 페어` 메뉴 클릭
* `키 페어 생성` 버튼 클릭
* 키 페어 이름 입력 후 `키 페어 생성` 버튼 클릭
* 자동 다운로드 된 *.pem 파일 저장

### 1단계 - 망 구성하기
#### VPC 생성
* `VPC` -> `VPC 생성` 버튼 클릭
* `VPC 이름` 입력
* `IPv4 CIDR` 입력
* `VPC 생성` 버튼 클릭
